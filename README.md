# 📝 해커톤중학교 창체의 날 누가기록 대체용 출결 확인 프로그램

구글 스프레드시트와 실시간으로 연동하여 학생의 출결 정보를 확인하고, 결석일의 활동을 자동으로 제외한 개인별 맞춤형 특기사항을 생성해 줍니다.
(과제물로 제외해달라고 말씀하신 생기부 작성이 아닌 엑셀 시트와 연결되는 출결 알림이 메인입니다.)

---

## ✨ 주요 기능

1. **실시간 구글 시트 연동 (출결 확인)**
   - Google Apps Script(GAS)를 활용하여 학급 출결 정보를 실시간으로 로드합니다.
   - 학생별 결석일 및 교시를 자동으로 감지합니다.

2. **결석일 활동 자동 제외 및 알림**
   - 학생이 결석한 날짜에 진행된 창의적 체험활동(자율활동) 내용은 특기사항 생성 시 자동으로 제외됩니다.
   - 담임 선생님이 인지할 수 있도록 결석일 활동 제외 안내 및 출결 확인 경고 알림을 표시합니다.

3. **개인별 맞춤형 특기사항 생성**
   - 등록된 총 15개의 활동과 30개의 다양한 문장 풀(Pool)을 활용하여 중복되지 않는 자연스러운 특기사항을 조합하여 생성합니다.
   - 생성된 텍스트의 글자 수(Byte)를 실시간으로 모니터링하여 나이스 입력 제한(한글 약 500자, 1,500 Byte)을 넘지 않도록 돕습니다.

4. **편리한 내보내기 기능**
   - 개별 복사 버튼을 제공합니다.
   - 전체 복사(엑셀용) 기능을 통해 학급 전체 학생의 특기사항을 한 번에 복사하여 엑셀에 행 단위로 편리하게 일괄 붙여넣기할 수 있습니다.

---

## 🛠 기술 스택

- **Frontend**: Vanilla HTML5, CSS3, JavaScript (ES6+)
- **Backend (API)**: Google Apps Script (Web App)
- **Database**: Google Sheets

---

## 🚀 사용 및 설정 방법

### 1. 구글 시트 연동 설정
프로그램 내부에서 구글 시트의 데이터를 읽어오기 위해 Google Apps Script API 주소를 사용합니다.
- `index.html` 파일 내부의 `GAS_URL` 변수에 배포된 웹 앱 주소를 설정합니다.
  ```javascript
  const GAS_URL = 'YOUR_GOOGLE_APPS_SCRIPT_WEB_APP_URL';
  ```

### 2. 실행 방법
- 별도의 빌드 과정 없이 `index.html` 파일을 더블 클릭하거나 로컬 서버(Live Server 등)를 통해 브라우저에서 바로 실행할 수 있습니다.
- 학년, 반, 학급의 맨 끝 번호를 입력하고 **"🔗 입력 및 연동하기"** 버튼을 누르면 연동이 완료됩니다.
